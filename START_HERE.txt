⚠️  IMPORTANT NOTES:
═══════════════════════════════════════════════════════════════════════════

Camera Access:
  ✅ Works: On local server (localhost) or GitHub Pages (HTTPS)
  ❌ Doesn't Work: Opening file directly (file://)

Browser Support:
  ✅ Best: Chrome, Edge (Chromium)
  ✅ Good: Firefox, Safari (iOS 13+)

Data Storage:
  • All photos stored in browser memory
  • Nothing uploaded until you manually upload ZIP
  • Settings saved in browser (not synced across devices)

═══════════════════════════════════════════════════════════════════════════
🆘 TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════════════════

Camera Not Working?
  → Use local server or GitHub Pages (not file://)
  → Check camera permissions in browser
  → Try Chrome if using another browser

ZIP Not Downloading?
  → Check download permissions
  → Disable popup blocker
  → Try a different browser

Settings Not Saving?
  → Check browser isn't in private/incognito mode
  → Clear cache and try again

More Help?
  → Open test.html to diagnose issues
  → Check browser console (F12) for errors
  → Read troubleshooting section in README.md
